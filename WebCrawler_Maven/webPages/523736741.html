Brett Cannon (brett@python.org) https://peps.python.org/pep-0685/ Tue, 08 Mar 2022 00:00:00 GMT PEP 684: A Per-Interpreter GIL https://peps.python.org/pep-0684/ Since Python 1.5 (1997), CPython users can run multiple interpreters in the same process. However, interpreters in the same process have always shared a significant amount of global state. This is a source of bugs, with a growing impact as more and more people use the feature. Furthermore, sufficient isolation would facilitate true multi-core parallelism, where interpreters no longer share the GIL. The changes outlined in this proposal will result in that level of interpreter isolation.
