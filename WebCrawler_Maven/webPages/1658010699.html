Qt para Python

O Projeto PySide2 tem como objetivo portar completamente o PySide para Qt 5.x. O desenvolvimento iniciou no GitHub em Maio de 2015. O projeto conseguiu portar o PySide para o Qt 5.3, 5.4 e 5.5. Durante o mês de abril de 2016, a The Qt Company decidiu fornecer apoio adequado ao projeto (ver detalhes).

Esta página wiki acompanha o progresso do desenvolvimento do PySide para Qt 5.x e fornece mais informações sobre o esforço.
