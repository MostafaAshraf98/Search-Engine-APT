IPython-dev -- IPython Community list

 

About IPython-dev
English (USA)

IPython provides a comprehensive set of tools for making effective use of Python when working interactively.

This is a unified, list dedicated to the discussion of usage and development of IPython. It supersedes the old ipython-{user, dev} lists previously hosted at Scipy.org.

While bug reports can be discussed here, it is strongly encouraged that you use IPython's issue tracker on github for that purpose.

http://ipython.org

To see the collection of prior postings to the list, visit the IPython-dev Archives.

Using IPython-dev
To post a message to all the list members, send email to ipython-dev@python.org.

You can subscribe to the list, or change your existing subscription, in the sections below.
