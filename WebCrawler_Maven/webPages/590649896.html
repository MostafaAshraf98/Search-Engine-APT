OpenStack Bare Metal

Provisioning and automating bare metal machines for containers and high performance computing.

Just in: The latest white paper from the OpenStack Bare Metal SIG
Read It Now 

Ironic allows users to manage bare metal infrastructure like they would virtual machines and provides ideal infrastructure to run container orchestration frameworks like Kubernetes to optimize performance.

At Verizon Media, OpenStack manages hundreds of thousands of bare metal compute resources with over 4 million cores in our data centers,” said James Penick, architect director at Verizon Media. “We have made significant changes to our supply chain process using OpenStack, fulfilling common bare metal quota requests within minutes.

James Penick
Architect Director, Verizon Media

From the perspective of cloud architects and administrators running Ironic, the service delivers support for a variety of use cases including:

Automating the entire server infrastructure ?lifecycle of deployments to updates ?to decommissioning.

Delivering cloud-like bare-metal infrastructure with multi-tenant networks to end users when used as a driver to OpenStack Nova.

With a standard API and lightweight footprint, Ironic can serve as a driver for a variety of bare metal infrastructure.
