Sponsor the Python Software Foundation

The Python Software Foundation raises and distributes funds to improve Python's packaging ecosystem.

Recent projects funded include:

  • The successful relaunch of the Python Package Index, powered by the new 'Warehouse' codebase
    With $170,000 in funding from the Mozilla Open Source Support Program in 2018
  • Improving PyPI's security and accessibility, and adding support for multiple locales
    With $80,000 in funding from the Open Technology Fund in 2019
  • Additional security-focused features for PyPI
    With $100,000 in funding from Facebook Research in 2019 and 2020
  • Overhauling pip's user experience and dependency resolver
    With $407,000 in funding from the Chan Zuckerberg Initiative and the Mozilla Open Source Support Program in 2020

With your support, the PSF can continue to fund packaging improvements, benefiting millions of Python users around the world.

Infrastructure sponsors

The following sponsors donate infrastructure that keeps PyPI up and running for millions of users across the globe

AWS

PyPI uses AWS infrastructure to host the machines that power our services, serve DNS, host our databases, and send and monitor email.

Visit AWS

Datadog

PyPI uses Datadog to collect metrics from the applications, services, and infrastructure behind the scenes allowing for the team to measure the impact of new changes, monitor for problems, and alert when systems fail.

Visit Datadog

Fastly

PyPI uses Fastly's CDN to quickly serve content to end-users, allowing us to minimize our hosting infrastructure and obscure possible downtime.

Visit Fastly

Google

Google is a visionary sponsor of the Python Software Foundation.

Visit Google

Pingdom

PyPI's infrastructure volunteers use Pingdom to monitor and receive alerts about downtime and other issues affecting end-users.

Visit Pingdom

Sentry

The PyPI team uses Sentry to capture, record, and respond to exceptions and errors on PyPI. Sentry is also a supporting sponsor of the Python Software Foundation.

Visit Sentry

StatusPage

The PyPI team uses StatusPage to communicate downtime, service degradation, or maintenance windows to end-users.

Visit StatusPage

Mozilla

In late 2017 the Python Software Foundation was awarded a Mozilla Open Source Support (MOSS) award.

This award was used to move PyPI from its legacy codebase and deploy Warehouse - the new codebase powering PyPI.

Visit Mozilla

Open Technology Fund

In 2019 the Python Software Foundation was awarded a contract through the OTF Core Infrastructure Fund.

This contract included an audit and improvements to the accessibility of PyPI, internationalization and translations for PyPI user interface, and security features including two-factor authentication and API tokens for uploads.

Visit Open Technology Fund
